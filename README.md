# Tilda Lander

A reimagined **Lunar Lander** for the [EMF Tildagon badge](https://tildagon.badge.emfcamp.org/).
Pilot a detailed spacecraft down through a field of rocks and set it down gently on a
glowing landing pad — land too fast, too steep, or off the pad and you'll crash.

## Controls

| Button            | Action                            |
| ----------------- | --------------------------------- |
| **LEFT / RIGHT**  | Rotate the lander                 |
| **UP** (or CONFIRM) | Fire the main thruster          |
| **CONFIRM**       | Start / advance / retry           |
| **CANCEL**        | Back to title, then exit the app  |

To land safely you must touch down on a pad with **both feet on the pad**, a **slow descent**, and the lander **close to level**. The on-screen `V` (vertical) and
`H` (horizontal) speed readouts turn green when you're within the safe envelope, and the tilt
bar shows how level you are.

## Features

- Detailed little spaceship.
- Atmospheric backdrop: a twinkling starfield, a ringed gas giant and a small moon.
- "Procedurally generated" rocky terrain with multiple landing pads each game.
- Pads are worth more the narrower they are (×1 / ×2 / ×3). Score rewards soft, centred,
  fuel-efficient landings.
- Increasing difficulty per level: stronger gravity, less fuel, rougher terrain, fewer pads.
- The 12 badge LEDs act as a live (green → yellow → red), flicker orange at the
  engine while thrusting, pulse green on a good landing and flash red on a crash.
- Screen shake and a particle explosion on impact.
- 3 lives, and a high score saved to the badge between sessions.
